Что такое SQL инъекция?
SQL инъекция - это уязвимость веб-безопасности, которая позволяет злоумышленнику вмешиваться в SQL-запросы, выполняемые приложением. Возникает когда данные пользователя неправильно валидируются и напрямую включаются в SQL-запрос.

Почему ' OR 1=1;-- позволил войти в аккаунт администратора?                        

OR 1=1 делает условие всегда истинным

;-- комментирует оставшуюся часть запроса

Запрос возвращает всех пользователей

Приложение использует первую запись (обычно администратор)

Как защитить web-ресурс от SQL инъекций?
